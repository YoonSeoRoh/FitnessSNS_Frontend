## Fitness SNS Frotend

### 프로젝트 설명
🏃‍♂️생활운동 리워드 적립 웹/앱 어플리케이션
- 자신의 운동량 체크 및 운동시 지역별로 리워드를 적립한다.
- 적립된 리워드를 제휴사 쿠폰으로 교환할 수 있다.
- 운동 인증을 통해 마이페이지에서 자신의 운동주기를 체크할 수 있고, SNS에 인증된 사진을 게시물로 등록할 수 있다.
- 화면 설계 : [화면설계]()
